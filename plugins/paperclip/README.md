# Paperclip — Onepilot plugin *(coming soon)*

This folder is the planned home for **Onepilot's Paperclip integration**.

[Paperclip](https://github.com/paperclipai/paperclip) is an open-source orchestration platform that turns AI agents into a *company* — org charts, budgets, governance, goal tracking (MIT license, ~38K GitHub stars in the first month after its March 2026 release).

## What this plugin will add

The first mobile surface for a Paperclip company:

- Live agent status cards — who's working on what, right now.
- Today's spend, per agent and rolled up.
- Queued and completed tasks.
- One-tap approvals when an agent is blocked on a human decision.
